# 24 — Product Roadmap 2026-2030

## Vision Reality
To transition Saarthi from an MVP (2026) to the default AI Operating System for ambitious individuals by 2030.

## 2026: Foundation & The Goal Engine (Current)
**Theme: Make Goal Tracking Intelligent**
- Launch the core Goal Engine.
- Implement robust Local AI fallback (Ollama).
- Launch Career and Learning modules.
- Basic Memory integration (Short-Term + Workspace RAG).
- *Success Metric*: Users successfully completing 3+ goals via Saarthi.

## 2027: The Agentic Leap
**Theme: Proactive AI**
- Transition from reactive chat to proactive agents.
- Agents can run background tasks (e.g., "Scrape job boards daily and apply to matches").
- Multi-modal memory (audio/voice notes integration).
- Deep integration plugins (Google Calendar, Notion, Linear).
- *Success Metric*: Agents saving users 10+ hours per week of manual work.

## 2028: The Knowledge Graph
**Theme: The Ultimate Second Brain**
- Upgrade Memory Engine to a continuous, personalized Knowledge Graph.
- Saarthi understands relationships: "The person you met last week works at the company you are applying to."
- Cross-module synthesis (Learning + Career + Research).
- Peer-to-peer Agent communication (Saarthi agents negotiating meetings with other users' agents).

## 2029: Ubiquity & Ambient AI
**Theme: Beyond the Screen**
- Ambient voice interfaces (always-on, wearble integrations).
- Fully autonomous execution of routine tasks.
- Advanced cost optimization allowing localized models to run entirely on mobile hardware.
- Saarthi becomes the central router for a user's digital life.

## 2030: The Personal OS
**Theme: Symbiosis**
- The concept of "apps" fades. Users interface primarily with Saarthi, which manages the APIs behind the scenes.
- Total privacy guarantees via confidential computing and zero-knowledge proofs.
- Saarthi is as essential as a smartphone.
